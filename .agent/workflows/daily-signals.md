# 每日买卖点信号系统

## 功能概述

系统会自动生成每日买入/卖出信号，基于以下条件：

### 买入信号条件

| 信号类型 | 条件 | 信心度 | 止损 | 目标 |
|----------|------|--------|------|------|
| BLUE突破 | BLUE > 180 | 70-95% | -5% | +15% |
| 趋势确认 | BLUE 150-180 + ADX > 25 | 50-70% | -7% | +12% |
| 黑马形态 | is_heima = True | 55% | -10% | +25% |
| 绝地反击 | is_juedi = True | 45% | -12% | +20% |
| 量价齐升 | 成交 > 50M + BLUE 120-160 | 55-75% | -6% | +10% |

### 卖出信号条件

| 信号类型 | 条件 |
|----------|------|
| 止损 | 当前价 <= 入场止损位 |
| 止盈 | 当前价 >= 入场目标价 |
| 超时卖出 | 持仓 > 10天 且 盈利 < 5% |

## 使用方法

### 1. 在 Streamlit 界面查看

1. 打开 https://coral-creek.streamlit.app
2. 进入 **📈 信号中心**
3. 点击 **🎯 今日信号** 标签
4. 侧边栏选择市场 (美股/A股)
5. 点击 **🔄 生成今日信号**

### 2. 通过 Telegram 接收

系统每日自动推送信号到 Telegram：
- 美股信号：美东 16:30 (收盘后)
- A股信号：北京 15:30 (收盘后)

### 3. 命令行手动运行
// turbo
```bash
cd versions/v3
python scripts/send_trading_signals.py --market US  # 美股
python scripts/send_trading_signals.py --market CN  # A股
python scripts/send_trading_signals.py --both       # 两个市场都发送
```

## 信号强度说明

- 🔥 **强烈**: 多条件共振，信心度 > 70%，建议重点关注
- ⚡ **中等**: 主要条件满足，信心度 50-70%，可适量参与
- 💧 **弱**: 单一条件触发，信心度 < 50%，仅供参考

## 持仓管理

信号追踪页面支持模拟持仓管理：

1. 添加持仓时记录：入场价、止损位、目标价
2. 系统自动追踪盈亏
3. 触发止损/止盈时生成卖出信号

## 注意事项

⚠️ **风险提示**:
- 所有信号仅供参考，不构成投资建议
- 务必设置止损，控制风险
- 建议分散持仓，单票不超过 10%
- 信号历史胜率仅供参考，不代表未来表现
